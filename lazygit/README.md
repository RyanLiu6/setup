# Lazygit

[Lazygit](https://github.com/jesseduffield/lazygit) is a simple terminal UI for git commands.

## Installation

Running `./setup` will install lazygit via:
- **macOS**: Homebrew (`brew install lazygit`)
- **Linux**: GitHub releases (latest version)

## Usage

Launch lazygit in any git repository:
```bash
lazygit
```

Key features:
- Stage/unstage files with visual feedback
- Interactive rebasing
- Cherry-picking commits
- Easy conflict resolution
- Stash management

## Configuration

This setup uses lazygit's defaults. Custom configuration can be added to:
- macOS/Linux: `~/.config/lazygit/config.yml`

See the [lazygit docs](https://github.com/jesseduffield/lazygit/blob/master/docs/Config.md) for configuration options.
